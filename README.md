
# Initial Setup:

Base requirements: Python 3, Pip, Git, Curl

Close this repository:
```
git clone https://github.com/joary/devgrid_python_test
```

Install requirements:
```
pip install -r requirements.txt
```

```
./initdb.sh
```

```
./run.sh
```
